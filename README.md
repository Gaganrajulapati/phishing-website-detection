# 🔐 Phishing Website Detection System

A machine learning web application that detects whether a website is phishing or legitimate using Random Forest.

## 🚀 Features

- Data visualization dashboard
- Feature importance analysis
- Real-time prediction
- Interactive Streamlit interface

## 🧠 Machine Learning

- Supervised Learning (Binary Classification)
- Random Forest Classifier
- Feature Selection & EDA

## 📊 Dataset

PhiUSIIL Phishing URL Dataset (UCI Repository)

## 🛠️ Technologies Used

- Python
- Scikit-learn
- Pandas & NumPy
- Streamlit
- Matplotlib & Seaborn

## ▶️ How to Run
